#Money-Changer#

Money-changer is a simple Wordpress plugin for displaying currency conversions to users using the [http://coinmill.com/](CoinMill) interface.

##Basic Usage##

[change_money from="USD" to="CAD,THB" amount="300"]Test[/change_money]

[change_money from="USD" to="CAD,THB" amount="300"]Test[/change_money]
